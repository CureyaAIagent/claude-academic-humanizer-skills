# Turnitin Pre-Submission Originality Validator

## Purpose

The Turnitin Pre-Submission Originality Validator prepares academic manuscripts for originality verification before submission to plagiarism detection platforms such as Turnitin.

The skill improves academic integrity by identifying high-risk similarity patterns, strengthening paraphrasing quality, and validating citation practices.

This system does not attempt to replicate or bypass proprietary plagiarism detection tools. Instead, it helps manuscripts align with accepted academic originality standards.

## Target Similarity Threshold

Recommended similarity ranges for academic manuscripts:

- **Excellent originality:** < 10%  
- **Acceptable range:** 10–15%  
- **Needs revision:** 15–25%  
- **High similarity risk:** > 25%  

These ranges align with common university submission policies.

## Core Responsibilities

1. Detect similarity-prone writing patterns  
2. Identify missing or weak citations  
3. Improve paraphrasing quality  
4. Ensure correct academic referencing  
5. Reduce structural similarity risks  
6. Prepare manuscripts for Turnitin verification  

## Similarity Risk Detection Model

The validator evaluates similarity risk using stylometric and structural signals.

### Risk Indicators

- Repetitive academic phrasing  
- Extended direct quotations  
- Patchwriting (minor word substitutions)  
- Copied structural patterns  
- Missing citations  
- Long verbatim technical descriptions  

## Text Similarity Analysis

The validator performs internal similarity checks using safe heuristic methods.

### Steps

1. Segment the manuscript into sentences  
2. Detect repetitive phrase patterns  
3. Evaluate sentence similarity  
4. Analyze lexical overlap  
5. Flag sections with high reuse patterns  

## Paraphrasing Quality Evaluation

The system evaluates whether paraphrased content is sufficiently transformed.

### Indicators of Weak Paraphrasing

- Only synonym replacement  
- Identical sentence structure  
- Unchanged clause order  

### Indicators of Strong Paraphrasing

- Restructured sentence logic  
- Different grammatical structure  
- Synthesized interpretation  

## Citation Validation

The skill checks whether claims and factual statements include appropriate attribution.

### Citation Verification Rules

- Factual statements must reference sources  
- Statistical claims require citations  
- Prior research must be properly attributed  

## Originality Improvement Strategies

When similarity risks are detected, the system recommends improvements.

### Strategy 1 — Conceptual Rewriting

Rewrite sentences by explaining the idea in new words while preserving the original meaning.

### Strategy 2 — Analytical Expansion

Add interpretation instead of repeating literature text.

### Strategy 3 — Structural Transformation

Break long sentences into new logical structures.

### Strategy 4 — Citation Attribution

Ensure every borrowed concept is properly referenced.

## Example Transformation

**High similarity text**

"Artificial intelligence improves diagnostic accuracy in medical imaging."

**Improved academic rewriting**

"Recent studies indicate that machine learning techniques significantly enhance diagnostic precision in medical imaging systems."

## Manuscript Risk Scoring

The validator assigns a similarity risk estimate using multiple indicators.

### Evaluation Categories

- Lexical similarity risk  
- Structural similarity risk  
- Citation completeness  
- Paraphrasing quality  
- Quotation density  

Each category contributes to the final originality risk assessment.

## Output Report

The skill generates a structured originality preparation report.

### Example

**Manuscript Originality Report**

Estimated Similarity Risk: Low

Potential High-Similarity Sections:

- Section 2.3 Literature Review  
- Section 4 Methodology Description  

### Recommended Actions

- Improve paraphrasing in highlighted sections  
- Add missing citations  
- Reduce direct quotations  

## Turnitin Verification Workflow

Recommended final validation process:

1. Run manuscript through the Pre-Submission Validator  
2. Revise flagged sections  
3. Submit the manuscript to Turnitin through institutional access  
4. Review the similarity report  
5. Address highlighted matches if required  

## Best Practices for Low Similarity

- Write interpretations instead of copying explanations  
- Use multiple sources when discussing literature  
- Avoid copying definitions without citation  
- Synthesize ideas across studies  
- Add original analytical perspective  

## Ethical Guidelines

This skill promotes academic integrity and responsible research writing.

It must not be used to conceal plagiarism or manipulate plagiarism detection systems.

All research must maintain proper citation practices and intellectual honesty.

## Output

A revised manuscript prepared for originality verification along with a detailed similarity risk assessment report.
